## ✨ Features

- 🎬 **10+ Cinematic Scenes** — countdown, matrix rain, galaxy fly, shockwave, torus spin, fire frame
- 🧮 **3D Engine** — Vec3, Camera с перспективной проекцией, вращение XYZ
- 🔥 **Heat System** — плавные градиенты от чёрного к белому через огонь
- 🎨 **RGB/HSV Colors** — полный контроль над цветовой палитрой
- ⚡ **Physics** — spring animations, gravity, particle systems
- 🎭 **ASCII Art** — правильный рендеринг VEXIS и 9Б КЛАСС

# Clone repository
